
## 免费好用的Mac软件
### [IINA 视频播放](https://iina.io/)
- 优点：专为 macOS 设计，界面美观，原生手势、Touch Bar 支持；基于 mpv，解码能力强；支持外挂/在线字幕、画中画、章节、HDR（取决于硬件/系统）等。
- 适合：日常观影、字幕党、追求 mac 体验的用户。

### [Transmission 磁力链下载](https://transmissionbt.com/)
- 特点：界面极简、资源占用低、原生支持 Apple Silicon。
- 适合：想“装上就用”、不折腾的人。
- 安装：Homebrew 用户可用命令 brew install --cask transmission
#### 使用
- 方式一：用系统“打开方式”
    1. 复制磁力链接（例如以 magnet:? 开头的那段）。
    2. 打开 Transmission。
    3. 菜单栏 File > Open URL…（或按 Command+U）。
    4. 粘贴磁力链接，点 Add/打开。
    5. 选择保存路径，确认。
- 方式二：直接粘贴到主窗口
    1. 复制磁力链接。
    2. 激活 Transmission 后，按 Command+V 粘贴，或点工具栏的“+”按钮再选择 “Open URL…”。
